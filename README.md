# Heart Disease Data Analysis

## Overview
This project analyzes a heart disease dataset using Python and Google Colab.

## Key Work:
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Data visualization

## Tools Used:
- Python
- Pandas
- Matplotlib / Seaborn

## Outcome:
Identified patterns and factors affecting heart disease risk.
